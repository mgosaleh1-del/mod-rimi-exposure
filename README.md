# اختبار ثغرات مواقع ويب

هذا المشروع يهدف إلى تطبيق عملي على بعض الثغرات الأمنية
.

تم التركيز في هذا المشروع على اختبار الثغرات بشكل يدوي
وملاحظة تأثيرها، ثم توثيق النتائج بطريقة بسيطة.

---


- تم اختبار مواقع تحتوي على ثغرات أمنية
- الهدف من العمل تعليمي فقط
- لا يحتوي المشروع على بيانات حقيقية

---

## الثغرات التي تم اختبارها
- Sensitive Data Exposure (Direct File Access)

---

## طريقة العمل
1. الدخول إلى الموقع
2. فحص الصفحات والروابط
3. اختبار الثغرات يدويًا
4. تصوير النتائج
5. توثيق الثغرات وطريقة استغلالها
